# Security Policy

## Supported Versions

| Version       | Supported          |
|---------------|-------------------|
| 1.x.x         | :white_check_mark: |
| < 1.x.x       | :x:               |

## Reporting a Vulnerability

If you find a security vulnerability, please email levikirkland@users.noreply.github.com or open a private security issue.

We will investigate and fix any issues as quickly as possible. Please do not use public issue trackers for security reports.

Thank you for keeping MyDigialLibrary safe!
